# Спринты и задачи для проекта "Лабиринт"

## Общее описание проекта

Проект представляет собой консольное приложение для генерации и решения лабиринтов. Приложение поддерживает два алгоритма генерации (DFS и Prim) и два алгоритма поиска пути (Dijkstra и A*). Также реализован графический интерфейс для визуализации лабиринтов.
